# Student Registration System

A comprehensive student registration system built with React that allows management of course types, courses, course offerings, and student registrations.

## Features

- **Course Types Management**: Create, read, update, delete course types
- **Courses Management**: Create, read, update, delete courses  
- **Course Offerings**: Associate courses with course types
- **Student Registration**: Register students for course offerings
- **Filtering**: Filter offerings by course type
- **Responsive Design**: Works on all device sizes
